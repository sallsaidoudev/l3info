Exception raised when the requested operation cannon be currently completed by the operating system
Error [Protocol]
Exception raised when information given or received does not match protocol

Exception raised when a connection is lost.

Quit [Input]
Exception Quit is raised when the player has requested quit either by pressing Ctrl-C, trying to close the window, or by non-fatal signal

Exception raised when the message is too short for the reader to read it's value

Exception raised when the given keyname is not known
